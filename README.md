## Installation

Add dependency to `pyproject.toml` of your Flet app:



* **Git dependency**

Link to git repository:

```
dependencies = [
  "flet-info-popup @ git+https://github.com/wanna-pizza/flet-info-popup",
  "flet>=0.28.3",
]
```

* **PyPi dependency**  

If the package is published on pypi.org:

```
dependencies = [
  "flet-info-popup",
  "flet>=0.28.3",
]
```

Build your app:
```
flet build macos -v
```
